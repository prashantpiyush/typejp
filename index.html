Type in Japanese

Hiragana / Katakana あア

How to use

Type in romaji and characters will automatically get converted.

Use lower case letters for Hiragana:

  • Typing 'shi' will produce - 'し'
  • Typing 'a' will produce - 'あ'

Use lower case letters for Katakana:

  • Typing 'SHI' will produce - 'シ'
  • Typing 'A' will produce - 'ア
